# movieDBservice

[주제]
영화관 티켓관리 Database CRUD로 구현하기

[소개] 영화관람에 관한 database를 관리하기 위한 서비스입니다. 관람자명, 좌석, 좌석 공석여부 등을 확인할 수 있습니다.


[필요한 메뉴(기능)] 
추가, 삭제, 조회, 수정, 파일저장/불러오기, user이름으로 검색, 영화명으로 검색, 상영관별 잔여좌석 확인
